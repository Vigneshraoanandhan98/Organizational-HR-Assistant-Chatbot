Organizational HR Assistant Chatbot

Developed an AI-powered HR Assistant Chatbot that streamlines employee support by providing instant responses to HR-related queries. The chatbot leverages Generative AI and Retrieval-Augmented Generation (RAG) to retrieve information from company policy documents and employee databases.

Key Features
📄 Company Policy Q&A using RAG
🏖️ Leave Balance and Leave History Tracking
📅 Attendance Monitoring and Reporting
👥 Employee Directory Search
🎫 HR Support Ticket Generation
🔐 Role-Based Access Control
💬 Natural Language Query Processing
Technologies Used
Python
SQL Server / MySQL
Streamlit
LangChain
FAISS Vector Database
Hugging Face Transformers
Large Language Models (LLMs)
RAG Architecture
Pandas
AWS Cloud Services
Business Impact

The solution reduces HR response time, improves employee self-service capabilities, and enables quick access to organizational information through an intelligent conversational interface.
